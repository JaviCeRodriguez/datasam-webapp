# Wiki del Proyecto DataSam WebApp

Esta carpeta contiene la documentación y guías del proyecto.

## Contenido

- **[Guia-de-Desarrollo-Local.md](./Guia-de-Desarrollo-Local.md)**: Guía completa para ejecutar el proyecto en tu computadora local, incluyendo:
  - Requisitos previos y configuración
  - Instalación paso a paso
  - Cómo contribuir al proyecto
  - Uso responsable de herramientas de IA
  - Solución de problemas comunes

## Cómo usar esta Wiki

Si eres nuevo/a en el proyecto, te recomendamos comenzar por la [Guía de Desarrollo Local](./Guia-de-Desarrollo-Local.md).

## Contribuir a la Wiki

Si encuentras errores o áreas de mejora en la documentación:

1. Crea un issue describiendo el problema o la mejora sugerida
2. Si deseas hacer los cambios tú mismo/a, sigue el proceso normal de contribución descrito en la guía de desarrollo local
